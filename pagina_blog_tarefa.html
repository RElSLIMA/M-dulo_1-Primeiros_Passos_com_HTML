    Tags

    O que é uma tag em HTML? Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

    Listas ordenadas e não ordenadas

    Listas ordenadas: Uma lista ordenada é também uma lista de itens. As listas de itens são marcadas com números. Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.

    Listas não ordenadas: As listas não numeradas são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas. As tags usadas para sua criação são <ul> e <li>. A tag <ul> é usada para definir a lista e a tag <li> é usada para cada item da lista.
